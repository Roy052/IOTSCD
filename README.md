    <h3> 주제 </h3>
     아두이노를 활용한 게임 제작 프로젝트
    <h3> 게임 플레이 </h3>
      
      두더지 잡기 : 시간 동안 튀어나오는 두더지를 마우스로 클릭한다.
      
      대탈출 : 장애물의 위치를 보고 방향을 맞춰 캐릭터를 출구로 보낸다.
      
      카드 뒤집기 : 같은 모양의 카드를 찾아 둘 다 뒤집는다.
      
      오셀로 : AI 혹은 1대 1로 오셀로 경기를 한다.
